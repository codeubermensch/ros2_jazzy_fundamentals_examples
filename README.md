# ROS2 Fundamentals Examples

This package contains examples demonstarting fundamental ROS2 concepts

## Description

This package includes minimalist ROS 2 code to demonstrated important ROS2 concepts and patterns
- Following ROS 2 Python style guidelines

## Prerequisites

- ROS 2 installed
- Python 3
- Created a ROS 2 workspace ('ros2_ws')
